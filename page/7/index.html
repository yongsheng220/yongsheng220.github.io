JSP

发表于 2022-01-20 分类于 java

简介

如果用servlet来向用户展示html页面，那么要用多个out.println打印html语句,非常不便，而且写个后端程序还得考虑html的打印，所以要将前后端分离，最后让后台产生的数据直接镶嵌到html多方便，这就是jsp的作用，但是说到底jsp还是一个servlet，比如创建一个index.jsp写入代码后，还是转为index_jsp.java 还是一个servlet，只不过自动添加了out.println(html)语句,然后调用servlet引擎，执行servlet类，产生html格式输出传递给容器，容器再传给客户端

数据库连接

四步

  • 导入JDBC包：使用Java语言的import语句在Java代码开头位置导入所需的类。

  • 注册JDBC驱动程序：使JVM将所需的驱动程序实现加载到内存中，从而可以满足JDBC请求。

  • 数据库URL配置：创建一个正确格式化的地址，指向要连接到的数据库(如：MySQL,Oracle和MSSQL等等)。

  • 创建连接对象：最后，调用DriverManager对象的getConnection()方法来建立实际的数据库连接。
