Docker CTF 出题记录

发表于 2021-08-10

前言

给下一届出题，记录一下docker的初步认识，不涉及sql

获取所有容器信息：
docker ps -a

搜索镜像
docker search lamp //搜索lamp环境直接就能用

拉取镜像
docker pull mattrayner/lamp

查看镜像
docker images

创建容器
docker create -p 4000:80 --name test-4000 mattrayner/lamp
//4000是访问的端口 test-4000是起的容器名字，就是别名

启动容器
docker start test-4000

启动成功

将题目复制到容器中
docker cp /var/www/html/test.php test-4000:/var/www/html

也可以进入容器中进行添加修改题目
docker exec -it ede4 bash //这里前提是容器名前几位没相同的
